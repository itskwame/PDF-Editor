# FreePDFFlow MVP (Minimal Starter)

This workspace contains a minimal Next.js starter implementing core routes from the PRD:
- `/` homepage
- `/edit` editor placeholder
- `/pricing` pricing placeholder

To install and build locally:

```bash
npm install
npm run build
npm start
```

This is a scaffold only — auth, PDF editor UI, Supabase, and Stripe are not implemented yet. Follow the PRD to continue implementation.
